Application development
#######################

.. contents::
    :local:
    :depth: 2

Bluetooth applications are developed using the common infrastructure and approach that is described
in the :ref:`application` section of the documentation.

Additional information that is only relevant to Bluetooth applications can be found on this page.

Thread safety
*************

Calling the Bluetooth API is intended to be thread safe, unless otherwise noted in the documentation
of the API function.
Bug reports and pull requests are welcome.
