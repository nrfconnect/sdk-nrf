Updating the firmware for nRF91 Series devices
##############################################

Updating the firmware for nRF91 Series devices involves several key steps to ensure the device operates with the latest features, improvements, and security updates.
The latest firmware binary compatible with the nRF91 Series device are available on the following pages:

* `nRF9151 DK Downloads`_
* `nRF9161 DK Downloads`_
* `nRF9160 DK Downloads`_

You can update the firmware using the Programmer app from `nRF Connect for Desktop`_ download.
