Building the application on nRF5340 DK with nRF7002 EK shield
=============================================================

.. toggle::

   To build the Matter over Wi-Fi |matter_type| variant on nRF5340 DK with nRF7002 EK shield attached, perform the following actions depending on the build tool:

   .. tabs::

      .. group-tab:: nRF Connect for VS Code

         1. When `building an application <How to build an application_>`_ as described in the |nRFVSC| documentation, follow the steps for setting up the build configuration.
         #. In the :guilabel:`Add Build Configuration` screen, click the :guilabel:`Add argument` button under the :guilabel:`Extra CMake argument` section.
         #. Add the following Kconfig options:

            .. code-block:: console

               -- -DSHIELD=nrf7002ek -DFILE_SUFFIX=nrf70ek -DSB_CONFIG_WIFI_NRF70=y -DCONFIG_CHIP_WIFI=y

      .. group-tab:: Command line

         Use one of the following options:

         * west with the following command:

         .. code-block:: console

            west build -p -b nrf5340dk/nrf5340/cpuapp -- -DSHIELD=nrf7002ek -DFILE_SUFFIX=nrf70ek -DSB_CONFIG_WIFI_NRF70=y -DCONFIG_CHIP_WIFI=y

         * CMake with the following command:

         .. code-block:: console

            cmake -GNinja -Bbuild -DBOARD=nrf5340dk/nrf5340/cpuapp -DSHIELD=nrf7002ek -DFILE_SUFFIX=nrf70ek -DSB_CONFIG_WIFI_NRF70=y -DCONFIG_CHIP_WIFI=y -DAPP_DIR=*app_path* *path_to_zephyr*/share/sysbuild
            ninja -C build
