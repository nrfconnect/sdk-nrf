Welcome to the |NCS|!
#####################

The |NCS| contains libraries and application examples for Nordic Semiconductor devices. It uses the |link_zephyr|_ as real-time operating system.

.. note::
   |NCS| v0.3.0 supports development with nRF9160 Cellular IoT devices.
   It contains references and code for Bluetooth Low Energy devices in the nRF52 Series, though development on these devices is not currently supported with the |NCS|.
