Universal Unique Identifiers (UUIDs)
####################################

.. contents::
   :local:
   :depth: 2

The UUID API provides Bluetooth host types and helpers for working with 16-bit, 32-bit, and 128-bit UUIDs.
It includes structures for representing UUID values and helper macros for declaring, initializing, and encoding UUIDs.
These APIs are used throughout the Bluetooth host stack to identify services, characteristics, descriptors, and other attribute types.
