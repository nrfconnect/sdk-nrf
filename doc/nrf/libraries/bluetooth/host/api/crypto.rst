Cryptography
############

.. contents::
   :local:
   :depth: 2

The Cryptography API provides Bluetooth host helpers for random data generation and encryption.
It includes helpers that use the Bluetooth controller for random number generation and AES block encryption.
It also provides AES-CCM encryption and decryption helpers for authenticated data processing.
