Type 2 Tag
##########

Use the Type 2 Tag library in the :ref:`nrfxlib repository <nrfxlib:nfc>` to enable NFC Type 2 Tags and configure the tag data.
:ref:`nrfxlib:type_2_tag` contains documentation for the library.

See the :ref:`ug_nfc` user guide for information about :ref:`ug_nfc_programming`.

Additionally, the |NCS| provides libraries for reading Type 2 Tags.
Note that these require an NFC polling device.
