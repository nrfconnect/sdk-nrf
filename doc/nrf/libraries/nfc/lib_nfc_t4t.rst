Type 4 Tag
##########

Use the Type 4 Tag library in the :ref:`nrfxlib repository <nrfxlib:nfc>` to enable NFC Type 4 Tags and configure the tag data.
:ref:`nrfxlib:type_4_tag` contains documentation for the library.


See the :ref:`ug_nfc` user guide for information about :ref:`ug_nfc_programming`.

Additionally, the |NCS| provides libraries for reading Type 4 Tags.
Note that these require an NFC polling device.
