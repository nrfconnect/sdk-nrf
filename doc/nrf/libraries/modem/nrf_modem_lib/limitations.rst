.. _limitations:

Limitations
###########

For a detailed description of limitations related to the modem firmware used,
including supported cipher suites for TLS/DTLS, refer to the changelog of that specific version.
The changelog can be found in the found in the `nRF9160 modem firmware zip file`_ or `nRF91x1 LTE firmware zip file`_ depending on the SiP you are using.
