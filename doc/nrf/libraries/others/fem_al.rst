.. _fem_al_lib:

FEM abstraction layer
#####################

.. contents::
   :local:
   :depth: 2

This library is a wrapper for the MPSL FEM.

Overview
********

The library simplifies low level usage of FEM hardware.

Configuration
*************

To enable this library, use the :kconfig:option:`CONFIG_FEM_AL_LIB`.
