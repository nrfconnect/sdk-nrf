.. _tests:

Tests
#####

|NCS| provides unit tests for some of its drivers and libraries to help you verify their functionality.
You can find them in the ``tests`` folder.

The tests are based on Zephyr Test Framework (Ztest).
For detailed information on how to run them, see :ref:`zephyr:test-framework`.

.. note::
   This documentation covers only a few selected unit tests.
