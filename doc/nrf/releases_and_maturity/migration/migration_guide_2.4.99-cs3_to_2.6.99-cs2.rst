Migration notes for |NCS| v2.6.99_cs2 for v2.4.99-cs3 users
###########################################################

This document describes the new features and changes implemented in |NCS| to take into consideration when migrating your application from |NCS| for the nRF54 customer sampling release v2.4.99-cs3 to |NCS| v2.6.99-cs2.

To ensure the nRF54H20 DK runs its components correctly, verify its compatibility with the |NCS| v2.6.99-cs2 release checking the version number on your DK's sticker.
The only versions of the nRF54H20 DK compatible with the |NCS| v2.6.99-cs2 are the following ones:

* Version PCA10175 v0.7.x (ES3)
* Version PCA10175 v0.8.0 (ES3)
