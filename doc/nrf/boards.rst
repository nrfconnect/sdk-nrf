Supported boards
################

Nordic boards are typically defined in the sdk-zephyr repository.
You can browse their documentation through the :ref:`Zephyr board catalog <zephyr:boards>`.

Some boards are defined in the sdk-nrf repository instead.
This section documents these boards.

.. note::

   Some of these board definitions may move to the sdk-zephyr repository in a future release.
