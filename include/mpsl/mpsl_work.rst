Multiprotocol Service Layer workqueue
#####################################

.. contents::
   :local:
   :depth: 2

The Multiprotocol Service Layer workqueue library makes it possible submit work to be processed by the :ref:`nrfxlib:mpsl` workqueue.

The Multiprotocol Service Layer workqueue is a workqueue running with a higher priority than the system workqueue.
