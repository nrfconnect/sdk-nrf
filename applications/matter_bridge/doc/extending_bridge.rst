Extending the application
#########################

The Matter Bridge application presents an example implementation that supports only a few bridged device types.
You can use the application as a reference for creating your own, but might need to extend or customize it to fulfill all your requirements.

This following subpages describe and introduce the modifications for their respective scenarios.
