Bluetooth: NRF Desktop
##########################

Overview
********

Nordic Semiconductor HID reference design.

Requirements
************

Supported board:
* PCA10056 (nRF52840 DK) - for mouse example,
* PCA20041 - for low-latency mouse reference design,
* PCA10059 - for USB dongle.
